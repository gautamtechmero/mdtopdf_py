## 1. Do's and Don'ts for Setting Up a Loyalty Program

| ✅ Do's | 🚫 Don'ts |
|---|---|
| Give points for visiting the store (check-in) | Give points per dollar spent on cannabis |
| Give points for referring a friend | Let members redeem points for free/discounted cannabis |
| Let members redeem points for accessories | Use punch cards ("Buy 10, get 1 free") for cannabis |
| Let members redeem points for branded merch | Advertise cannabis discounts on social media or window signs |
| Offer Member-Only Pricing (lower menu price for members) | Send promo emails/texts without customer consent |
| Send push notifications for point balance updates via Apple/Google Wallet | Send push notifications promoting cannabis products or strains |
| Send push notifications for new merch/accessory drops | Send push notifications with cannabis pricing or discounts |
| Send push notifications for reward redemption reminders | Send push notifications with "Buy now" or urgency language for cannabis |
| Send push notifications for loyalty tier upgrades or milestones | Send push notifications to members who have not opted in |
| Update digital card with current point balance and member tier | Send push notifications that display cannabis imagery or branding visible on lock screen to minors |
| | Show cannabis purchase history or spending data on the digital card |

### Key Patterns Across All Brands

- **Every brand uses a digital/app-based system** — no paper punch cards
- **No brand gives points per dollar spent on cannabis** — all use check-ins or visit-based earning
- **Redemption is always for non-cannabis items** — accessories, merch, experiences
- **Member-only pricing** is the most common way to give members a benefit on cannabis itself

### Canna Cabana's "Market vs Elite" Pricing — Is It Actually Legal?

Canna Cabana shows **two prices side by side** on their website: a "Market" price (for non-members) and a lower "Elite" price (for $35/year paid members). For example:

| Product | Market Price | Elite Price | Difference |
|---|---|---|---|
| Pink Kush PR 2x1g | $11.99 | $8.68 | ~28% lower |
| Drippyz Bubba Joon Infused PR | $40.99 | $31.08 | ~24% lower |
| Bluenose Berry PR 5x0.5g | $21.99 | $16.24 | ~26% lower |

**This is a legal grey area — not clearly legal.** Here's both sides:

#### The Problem — Section 24 of the Cannabis Act

Section 24 of the federal Cannabis Act **prohibits providing any thing or service as an inducement for the purchase of cannabis.** Charging a $35/year fee that **directly results in 24–28% lower cannabis prices** can be argued as:

| Argument Against (Why It May Violate Section 24) |
|---|
| The $35 fee is essentially buying a "coupon" that lowers the price of a controlled substance |
| It creates a financial incentive to buy more cannabis to "earn back" the $35 membership fee |
| Health Canada has stated that offering "very low" sale prices can itself be treated as an inducement |
| The primary visible benefit of the paid tier IS cheaper cannabis — that looks like a commercial promotion |
| It locks consumers into a purchasing pattern with a controlled substance (pay fee → buy more to justify fee) |

#### Why Canna Cabana Has Gotten Away With It (So Far)

| Reason | Explanation |
|---|---|
| Scale and legal resources | High Tide Inc. (Canna Cabana's parent) is a publicly traded company with a large legal team that has structured the program carefully |
| "Costco defence" | They position it as a wholesale membership club, not a cannabis discount card. The $35 fee "buys access to the club," not a discount on weed |
| Non-cannabis perks bundled in | The Elite tier includes delivery discounts, birthday bags, events, and giveaways — so the fee is not ONLY for lower cannabis prices |
| AGCO hasn't ruled specifically on it | The AGCO has not publicly endorsed or penalized this exact model. They have penalized other practices (e.g., Cannabis Xpress was fined $200,000 for LP pay-to-play schemes), but haven't gone after paid membership pricing yet |
| Free tier also has member pricing | Because the free Cabana Club also offers member pricing on cannabis, they can argue the paid tier is about the extra non-cannabis perks, not the cannabis price |

## 3. How to Implement a Membership-Based Loyalty Program

### The Canna Cabana Model (Canada's Most Successful Example)

Canna Cabana, operated by **High Tide Inc.**, runs the largest cannabis loyalty program in Canada with over **1.5 million members**. Here's exactly how they do it:

### Their Two-Tier Structure

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   TIER 1: CABANA CLUB (Free)                        │
│   ─────────────────────────                         │
│   • Free to join                                    │
│   • Instant access to lower member prices           │
│   • Earn check-in points                            │
│   • Redeem points for accessories                   │
│                                                     │
│   ─────────────── UPGRADE ──────────────            │
│                                                     │
│   TIER 2: CABANA ELITE (~$35/year)                  │
│   ─────────────────────────                         │
│   • Everything in free tier, PLUS:                  │
│   • Deep discounts on delivery fees                 │
│   • Exclusive VIP event invitations                 │
│   • Birthday accessory bag                          │
│   • Raffle entries (Tesla giveaway, etc.)            │
│   • Flash sales on limited lifestyle gear           │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### How VIP Cannabis Can Copy This Model with Happy Rewards

#### Step 1 — Launch the Free Tier First

| What to Set Up | Details |
|---|---|
| **Name** | VIP Rewards (Green Tier) |
| **Cost** | Free |
| **Sign-Up** | In-store (tablet/kiosk) + Happy Rewards app + VIP Cannabis website |
| **Age Check** | 19+ ID verification in-store; date-of-birth gate online |
| **What Members Get** | Member-only cannabis pricing + earn check-in points + redeem for accessories/merch |

#### Step 2 — Add the Paid Tier After 3–6 Months

| What to Set Up | Details |
|---|---|
| **Name** | VIP Rewards (Purple Tier) |
| **Cost** | $9.99/month |
| **Who Can Join** | Only existing Green Tier members can upgrade |
| **Cannabis Pricing** | Same as Green Tier — NO extra cannabis discount |
| **Paid Perks (Non-Cannabis Only)** | Free delivery on all orders |
| | 20% off accessories (papers, pipes, vaporizers) |
| | Exclusive limited-edition VIP Cannabis merch drops |
| | Priority access to education events and workshops |
| | Monthly raffle entry (electronics, gift cards, lifestyle prizes) |
| | Enhanced birthday package (premium accessory kit) |
